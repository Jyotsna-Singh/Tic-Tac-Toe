# Tic-Tac-Toe

This is a very basic **Tic-Tac-Toe** game.It will implement a few rules and conditionals for things like not being able to click in the same square twice.

* Features/Technologies: 
  * HTML5, CSS3 
  * jQuery, JavaScript
  * Conditionals
